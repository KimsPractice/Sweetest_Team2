## 개발 환경

Express, Socket.io, Crome browser

- babel: ES6 문법 사용하기위한 트랜스파일러
- nodemon: JS파일 내용 변경시 자동 서버 재시작
- pug: 프로토타입용 뷰 엔진. (sia 추가)
